基本概念

语法

区分大小写

ECMAScript中的一切（变量、函数名和操作符）都去区分大小写，这意味着变量名test和变量名Test分别表示两个不同的变量，而函数名不能使用typeof，因为他是一个关键字。

标识符

定义：指变量、函数、属性的名字或函数的参数。
规则：
第一个字符必须是字母、下划线（_）或一个美元符号（$）;
其他字符可以是字母、下划线、美元符号或数字。
标识符一般采用驼峰大小写格式，如：
myCar

注释

  • 单行注释
    两个斜杠 //

  • 多行注释
    以一个斜杠和一个星号（/*）开头，以一个星号和一个斜杠（*/）结尾，如下所示：
    /*
    *这是一个多行（块级）注释
    *
    */
    中间加两个星号是为了提高注释的可读性。

    严格模式

    “use strict”;
    严格模式为js定义了一种不抬的解析与执行模型。站在严格模式下，ES3的一些不确定的行为将得到处理，而且对某些不安全的操作也会抛出错误。

    关键字和保留字

    关键字和保留字不能用作标识符，其中全部关键字：
    break do instanceof typeof
    case else new var
    catch finally return volid
    continue for switch while
    debugger function this with
    default if throw
    delete in try
    保留字:
    abstract enum int short
    boolean export interface static
    byte extends long super
    char final native synchronized
    class float package throws
    const goto privete transient
    debugger implements protected volatile
    double import public

    数据类型

    ES有5种简单数据类型（也称为基本数据类型）：Undefined、Null、Boolean、Number和String。
    还有一种复杂数据类型——Object,Object本质上是由一组无序的明值对组成的。
