作用域

函数作用域(局部作用域)

定义：在函数内部让这个变量屏蔽在函数内部，在函数内部做的事情在函数外部得不到任何信息。

解析：console.log(a)控制台显示a is not defined，因为test函数的函数作用域起作用了，它把变量a屏蔽在了函数test的大括号里面，在外面都拿不到。
思考：有的时候希望函数作用域的变量屏蔽在函数内部，有的时候希望共享，在外面也能够使用，这个时候该怎么办？
（1）像上面一样使用return向外面传递a的值；
（2）使用闭包的方式向外传递

调用test函数最终返回的是test2函数，而test2函数返回的结果就是7，其中a找不到就向上一级作用域找，找到a为3，b在它本身内部，所以返回的是7。
如果在test2内部return的是a + b + c,c找不到就会向上找，它的上一级是test,找不到继续向上到windows去找，最后也没有，会显示c is not defined。

var与let的区别

使用let声明的变量不可以通过window进行访问，而使用var声明的变量则可以。

使用let声明的变量不允许重复声明，使用var的则可以。
总结：
（1）使用let声明的变量具有块状作用域；
（2）let声明的变量不能通过全局对象的属性进行访问；
（3）let不能重复定义一个变量；
（4）let声明的变量不会变量提升。

const

const声明的变量不能够先声明再赋值，let和const可以
